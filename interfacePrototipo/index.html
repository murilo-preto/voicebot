Interface para Usuário Remoto VOICEBOT.

O programa python está disponível no botão download abaixo.

Instruções:

Pacotes necessários para executar:

  • Tkinter
  • SpeechRecognition
  • Playsound 1.2.2


    • Comandos do instalador pip:
    > pip install tk
    > pip install SpeechRecognition
    > pip install playsound==1.2.2
